skenario 1: Kode ini memanfaatkan CSS Grid untuk menyusun elemen dalam tiga kolom otomatis menggunakan grid-template-columns: auto auto auto, dengan jarak antar item sebesar 50px melalui grid-gap. Kelas .tempat_grid dan .item_grid menambahkan gaya seperti warna latar dan padding untuk tampilan yang rapi. Kesimpulannya, CSS Grid adalah alat yang sederhana namun powerful untuk membuat tata letak yang terstruktur, fleksibel, dan mudah disesuaikan.
skenario 3:Properti grid-column-gap: 100px mengatur jarak horizontal sebesar 100px antar kolom dalam grid. grid-row-gap: 50px menentukan jarak vertikal sebesar 50px antar baris. Keduanya meningkatkan ruang antar elemen grid untuk tata letak yang lebih terbaca.
skenario 4:Properti grid-gap: 50px mengatur jarak sebesar 50px baik antar kolom maupun baris dalam grid secara seragam. Ini adalah shorthand untuk grid-column-gap dan grid-row-gap, menyederhanakan penulisan dan meningkatkan keterbacaan tata letak.
