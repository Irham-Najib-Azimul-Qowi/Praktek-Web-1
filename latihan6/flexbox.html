Kesimpulan skenario 1: dengan display: flex membuat iten angaka dalam div menjadi baris baru

Kesimpulan skenario 2: degan tambahan style pada skenario mengubah item item nya memilikibackground,margin 10px, padding 20px,

kesimpulan skenario 3: menambahkan flex-direction: flex yang mengubah tampilan item item menjadi kolom

kesimpulan skenario 4: singkat saja Properti nowrap pada CSS (white-space: nowrap) mencegah teks membungkus ke baris baru, menjaga konten dalam satu baris. Cocok untuk menu atau label, tetapi perlu dikombinasikan dengan pengaturan overflow agar responsif dan rapi.

kesimpulan skenario 5: flex-flow: row wrap menyusun item flex secara horizontal (row) dan membungkus ke baris baru (wrap) jika ruang habis, cocok untuk tata letak responsif.

kesimpulan skenario 6: justify-content dalam CSS flexbox mengatur distribusi item di sepanjang sumbu utama (horizontal pada row, vertikal pada column). Nilai seperti flex-start (kiri/atas), flex-end (kanan/bawah), center (tengah), space-between (ruang merata antar item), dan space-around (ruang merata di sekitar item) menentukan posisi dan jarak. Singkatnya, ini mengontrol penyebaran item flex secara efisien dalam kontainer.
