Uji coba 4



Bisa memilih warna dengan tag input dengan tipe color, kemudian dapat memilih tanggal dengan tag input tipe date, dan pada tag input dengan tipe button dapat menampilkan pesan peringatan.

dengan tag input dengan tipe datetime-local kita bisa menginput tanggal beserta dengan waktu lokal, kemudian dengan tag input dengan tipe month kita bisa menginput bulan, dan untuk tag input dengan tipe email kita bisa menginput alamat email.

Uji coba 6




Fungsi dari tag input bertipe file yaitu untuk mengupload file dari device pengguna.

Uji coba 7





Uji coba ini menunjukkan bahwa atribut min dan max pada input number dan range membatasi nilai yang dapat dimasukkan. Input number memungkinkan pengguna mengetik angka, tetapi nilai di luar batas bisa ditolak. Input range hanya memungkinkan pemilihan dalam rentang yang ditentukan. Fitur ini membantu mengontrol input dan mencegah kesalahan pengguna.

Uji coba 8




Uji coba ini menunjukkan bahwa input url menerima alamat web, search digunakan untuk pencarian, dan tel untuk nomor telepon. Setiap tipe memiliki fungsi khusus yang membantu validasi dan kemudahan pengisian formulir.

Uji coba 9



Uji coba ini menunjukkan bahwa input time memungkinkan pengguna memilih waktu, sementara input week memungkinkan pemilihan minggu dalam satu tahun. Keduanya mempermudah input data waktu secara akurat.
